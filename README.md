## Moustic Reunion
Application Android. <b>Clé dichotomique d'identification morphologique des moustiques de l'île de LA REUNION.</b>
Télécharger l'application .apk sur votre telephone Android (par exemple dans votre dossier "download"), et installez la.
Ne pas tenir compte des messages d'avertissement lors de l'installation, puisque le fichier .apk est téléchargé d'une source différente du "Play Store".
La clé est en français.
Plus de détails sur cette application sont dispoonibles sur ZENODO.
