# entomo_tools
Applications and scripts for vectorial entomology

Les applications et scripts disponibles sont les suivants:

## Moustic Reunion
Application Android. Clé dichotomique d'identification morphologique des moustiques de l'île de LA REUNION.
<p>Télécharger l'application .apk sur votre telephone Android (par exemple dans votre dossier "download"), et installez la. 
Ne pas tenir compte des messages d'avertissement lors de l'installation, puisque le fichier .apk est téléchargé d'une source différente du "Play Store".
La clé est en français.
Plus de détails sur cette application sont dispoonibles sur ZENODO
